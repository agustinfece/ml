So what's this all about?

This machine learning portfolio is a collection of completed independent projects and interesting findings, each of which uses machine learning in certain way. It also tries to illustrate, in a friendly way, the basic concepts of machine learning for those who do not know and want to be informed.

Honestly, this is part of a course evaluation and that is the sense of being for this portfolio. After saying that, I hope it is helpful and enjoyable for the reader.
