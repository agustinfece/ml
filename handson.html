CRISP-DM Model

Cross-Industry Standard Process for Data Mining (CRISP-DM) is a comprehensive data mining methodology and process model that provides anyone from novices to data mining experts with a complete blueprint for conducting a data mining project.
CRISP-DM breaks down the life cycle of a Data Mining project into six phases. The idea is not to get into these concepts, just introduce them to put them into practice. Below are the main tasks that belong to each phase.
