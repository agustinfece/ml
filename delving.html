Delving Into The Concept

The concept of machine learning can be rigorously enlarged, but it escapes from the objective of this portfolio. Here it is intended to provide an approach as straightforward as possible. So here it goes my own definition:

Definition

Machine learning: an application of artificial intelligence that creates self-learning algorithms which learn from data and improve from experience, without being explicitly programmed.

Introduction

There is a single principle that underlies all these algorithms. What they all intend is to learn a desired function, which best approximates mapping an input to an output. The most common approach is to learn the mapping function to make predictions for new input, this concerns to a Machine Learning subfield called Predictive Modeling.
These algorithms could be grouped into Supervised and Unsupervised, Semi-supervised could be a hybridization of them.

Supervised ML

Supervised algorithms infer a function from labeled training data. This means that each instance contains the desired output value, the correct answer. They can be further grouped into:
Classification: separates inputs into categorical classes.
Regression: involves separating input into output real values.

Unsupervised ML

In this case, without predefined labels, the learning process is guidanceless. The algorithm by itself discover and present the interesting structure in the data. They can be further grouped into:
Clustering: groups a set of instances, in such way that the ones in the same group are more similar to each other than to those in other groups.
Association: discovers interesting relations or rules between variables.

Concept Discernment

There are some areas of study, which may feel identified with the definition I have just given. Thats why its important to differentiate them from what Machine Learning is. Each of those can be as developed as Machine Learning, but here they will just be introduced.

Artificial Neuronal Networks

Inspired by biological neural networks, artificial neurons are interconnected as a net with the purpose of learn to identify characteristics from the examples presented to them.

Deep Learning

It is a sort of Machine Learning algorithm which uses multiple layers of Artificial Neural Networks to, step by step, extract higher level features from the input.

Data Mining

Set of techniques and technologies that allow obtaining significant, implicit, previously unknown and potentially useful information from large volumes of data.

Statistical Analysis

Discipline that collects, explores and presents data with the objective of identify and reveal patterns and trends.

Cognitive System

A system that goes through an intellectual process in which it gains knowledge and can modify its behavior based on experience.
