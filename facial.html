Moravec Paradox

In the 1980s, Hans Moravec, Rodney Brooks, Marvin Minsky and others articulated and discussed this AI paradox. It's a paradox because for AI the complex is easy, and the easy is complex. As Moravec put it: “It is comparatively easy to make computers exhibit adult level performance […] and difficult or impossible to give them the skills of a one-year-old.” One of these mentioned skills is facial recognition. There are four steps to achieve it: First, a face must be found in the image. Second, facial features are analyzed. Then, they are compared against known faces. And last, the prediction is made.

Facial Recognition

The technology behind is Deep Learning. One might think that the way to achieve it is performing classification, making a neuronal network learn to predict if the face it has seen it’s the users’s one or not. So, it would use some training data to predict “true” or “false”, but here this approach would not work. This would mean, first, to re-train the network from scratch using the new obtained data from the user’s face. Second, once the user owns the phone, this would require a lot of time, energy consumption, and impractical availability of training data of different faces to have negative examples. It is believed that this should be driven by a neuronal network which can learn to compute distances between particular kind of data, such as images. Once face data is obtained, the network will learn to extract the most meaningful features from data, and compress it into n-dimensional array. Then, it is trained to make this mapping so that data points from different classes are as far as possible, while data points from the same class are as close as possible.




Animoji

This face-tracking technology allows to produce face expressions into emojis aspect. Animoji is available for an iPhone X or later, or an iPad Pro 11-inch or iPad Pro 12.9-inch (3rd generation) or later. The emojis available for this feature are Alien, Bear, Boar, Cat, Chicken, Cow, Dog, Dragon, Fox, Ghost, Giraffe, Koala, Lion, Monkey, Octopus, Owl, Panda, Pig, Pile, Rabbit, Robot, Shark, Skull, Tiger, T-Rex and Unicorn.
